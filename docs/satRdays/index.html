How Can Power BI Help Prepare Your Data?

  • Data Connectivity & Ingestion: Access data stored in hundreds of data sources through a clickable interface
  • Data Transformation: Apply over 350 different data transformations, preview data, and see history of applied steps, all through the user interface
  • Data Query Extensions: Comes with its own highly-intuitive data mashup language that can be customized to help with any data preparation activity
